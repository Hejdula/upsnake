# Multiplayer Game with Reconnection

This project consists of a C++ server and a Python client using a text-based TCP protocol for simple Multiplayer snake game.

## Prerequisites

- C++ Compiler (g++)
- Python 3
- PyQt6 (`pip install PyQt6`)

## Building the Server

Navigate to the `server` directory and run `make`:

```bash
cd server
make
```

## Running the Server

```bash
./server
```

## Running the Client

Navigate to the `client` directory and run `client.py`:

```bash
cd client
python3 client.py
```
